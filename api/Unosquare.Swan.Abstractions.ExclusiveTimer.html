    Class ExclusiveTimer

    A threading Unosquare.Swan.Abstractions.ExclusiveTimer._backingTimer implementation that executes at most one cycle at a time in a ThreadPool thread. Callback execution is NOT guaranteed to be carried out on the same ThreadPool thread every time the timer fires.
